## ✨ Key Features

This portfolio is built with a focus on buttery-smooth performance, modern UI/UX principles, and non-blocking 3D rendering.

* **Global WebGL Context:** Utilizes a persistent, fixed-background Three.js canvas featuring a custom, scroll-reactive magnetic particle swarm.
* **Cinematic Animations:** Staggered terminal-style typography and seamless scroll-triggered reveals powered by Framer Motion.
* **Glassmorphism 3D Project Cards:** Custom-built React Three Fiber components that leverage `useTexture` and interactive mouse-tracking for a tactile, frosted-glass hover effect.
* **Modular Architecture:** Adheres to strict separation of concerns, keeping 3D canvases, HTML layout sections, and data sources neatly isolated for a scalable codebase.
* **Fully Responsive:** Optimized for all screen sizes, ensuring WebGL effects scale gracefully without dropping frames on mobile devices.
